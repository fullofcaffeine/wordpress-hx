# Execution Plan Template

Use Beads for task state. This file is only a reusable shape for large task plans.

## Goal

State the Beads issue ID, PRD external reference, and intended verified outcome.

## Inputs

List upstream refs, manifests, fixtures, and receipts required before work begins.

## Implementation

Describe the bounded workset, owned paths, public interfaces, and compatibility constraints.

## Verification

List exact commands, oracle comparisons, receipts, and manifest updates required before close.

## Handoff

Record discovered work as Beads issues and summarize any remaining risks.
